# Privacy Policy

**Git repository List** is an Azure DevOps extension published by CrazySolutions.

## Data collection

This extension does not collect, store, or transmit any personal data to any external server or third party.

## How the extension works

The extension reads your current Azure DevOps session identity (your login name) and queries the Azure DevOps Git Refs API (`vso.code` read-only scope) to list all repositories that you have access to. All processing happens entirely within your browser and your Azure DevOps organisation. No data leaves your Azure DevOps environment.

## Telemetry and analytics

The extension does not include any telemetry, crash reporting, or analytics.
